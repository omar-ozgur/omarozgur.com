Software

BrainSpace VR

BrainSpace VR is a program that allows users to manipulate a 3D brain model in virtual reality (using the Oculus Rift and Myo armband). Users can select areas of the brain using their fingers in order to see information from the internet.

Globalize

Globalize is an app that allows users to point at a city in virtual reality (using the Oculus Rift and Leap Motion sensor), and find data about that city from Yelp, Twitter, Weather sources, and Wikipedia.

Ocular Trauma Score

I developed an app for Android and iOS devices that helps ophthalmologists diagnose eye conditions based on a calculated "ocular trauma score"

Anagram Finder

I created this program to quickly anagrams in a dictionary that contains thousands of words.

3D Graphs

I built this application during my first hackathon at UCLA. It allows for users to create simple graphs in a 3D environment, and then walk around those graphs from a first-person perspective. For example, the bars in a bar graph are solid, 3D objects.
